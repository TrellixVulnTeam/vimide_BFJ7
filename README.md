An automatic configuration program for vim
===============================================i

## install
### Ubuntu

- 版本要求

`ubuntu16.04`及其以上系统。

- 安装vimplus(建议在普通用户下安装)

		```bash
		mv ~/.vim ~/.vim.backup
		cp ~/.vimrc  ~/.vimrc.backup 
		git clone https://github.com/gaoyubin/vimide.git ~/.vim
		cp ~/.vim/vimrc  ~/.vimrc   
		```
